Overview
--------

This charm provides a Virgo based Instauth subordinate charm.  

Usage
-----

Deploy the charm as follows:

    juju deploy instauth

Configuration
-------------

The charm is configured via its embedded Virgo buildpack (in `vendor/virgo-buildpack`).
